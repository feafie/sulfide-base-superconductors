# Sulfide-based superconductors dataset

This repository provides supporting data for the manuscript:

**High-throughput discovery of ternary pyrite-type sulfide superconductors by metal-site substitution and pressure tuning**

## Files

* `raw-dataset.zip`
  Raw dataset used in this work.

* `POSCAR_of_Superconductor.zip`
  POSCAR files of six reported sulfide superconductors.
